MessageUp.js
github.com/aka-jain/MessageUp.js

MessageUp.js is just a 1KB Javascript library which runs without using any third party plugin.
If you want to change your text dynamically after some time then MessageUp.js will do it all for you.
You can also customize speed, color and effects time of messages.

Read documentation on GitHub.
And if you want to contribute in this library then you are most welcome.

It's very easy to apply.

Step1- include message.js or message-min.js script after your ending body tag like:
❮script src="/path/messageup.js"❯❮/script❯ or ❮script src="/path/messageup-min.js"❯❮/script❯

Step2- initialize your own script just above your message.up script tags as
❮script❯
     var msg=["Message 1", "Message 2", "Message 3", "Message 4", "Message 5" ];  //define messages here
     var color=["red", "green", "yellow", "green", "blue"];  // define color here (optional)
     var time=1500;  // define time here (optional)
     var speed="slow";  // define speed here as slow or fast (optional)
     var iteration="repeat";  // define iteration type here as repeat or no-repeat (optional)
❮/script❯

Step3- include an id to your div as message-box in HTML like:
❮div id="message-box"❯❮/div❯

Now you are ready to go :)
